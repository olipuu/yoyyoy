# Проект по предмету ТРПО

Это учебный репозиторий для выполнения заданий по дисциплине ТРПО.

## Цели
- Освоить базовый рабочий процесс Git и GitHub.
- Научиться работать с ветками, коммитами и pull request (PR).
- Организовать задачи в Project board (Projects).

## Структура
- README.md — описание проекта и целей.

## Как работать с проектом
1. Создайте новую ветку для изменений.
2. Внесите правки и закоммитьте.
3. Отправьте ветку в удаленный репозиторий.
4. Откройте pull request (PR).
